<all-roles> ​

Usage ​

  1. If you haven't already, go through the setup for the module.

  2. Import the <all-roles> element somewhere in the javascript side of your web-app like this:

js
import '@darksoil-studio/roles/dist/elements/all-roles.js'
  1. Use it in the html side of your web-app like this:
html
<all-roles>
</all-roles>

WARNING

Like all the elements in this module, <all-roles> needs to be placed inside an initialized <roles-context>.
